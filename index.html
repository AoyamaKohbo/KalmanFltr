Popovの超安定論

Popovの超安定論は図に示す線形定常ブロックと非線形時変ブロックで構成されるフィードバックシステムが次の二つの条件を満足するとき、そのフィードバックシステムは安定(超安定)であることを示しています。

・線形ブロックの伝達関数が強正実条件を満足する。

・Popovの積分不等式が成立する。

強正実条件:

線形ブロックの伝達関数\(G(s)\)が次の条件を満足することです。

(1)\(G(s)\)が実のsに対して実である。

(2)\(G(s)\)が漸近的に安定、すなわちs面の右半平面\(Re[s]\ge0\)に極がない。

(3)\(-\infty <\omega< \infty\)のすべての\(\omega\)に対して\(Re[G(j\omega)]\ge0\)。

Popovの積分不等式:

線形ブロックの出力を\(v\)、非線形ブロックの出力を\(w\)として、Popovの積分不等式は次式で与えられます。

\begin{align} \int_{0}^{t}vw dt \ge -\gamma_0^2 \end{align}

ここに、\(\gamma_0\)は定数です。

それでは、具体的なフィードバックシステムについて考察してみましょう。

\begin{align} \ddot x + a\dot x + b x = u \end{align}

これを少し変形して、

\begin{align} \ddot x = - a\dot x - b x + u,~~~\dot x = \dot x. \end{align}

さらに行列とベクトルの形式で記述します。

\begin{align} \begin{bmatrix} \dot x \\ \ddot x \end{bmatrix} = \begin{bmatrix} 0 & 1\\ -b & -a \end{bmatrix} \begin{bmatrix} x \\ \dot x \end{bmatrix} + \begin{bmatrix} 0 \\ 1 \end{bmatrix}u \end{align}

行列とベクトルを次のように太字で表します。

\begin{align} \bf x = \begin{bmatrix} x \\ \dot x \end{bmatrix},~~~~~ \bf A = \begin{bmatrix} 0 & 1\\ -b & -a \end{bmatrix},~~~~~ \bf B = \begin{bmatrix} 0 \\ 1 \end{bmatrix}. \end{align} \begin{align} \dot{\bf x} = \bf{A} \bf{x} + \bf{B}u \end{align}
